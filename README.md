# JOB-A-YO_PJ
> 핵심 기능 : 사용자가 원하는 소비층, 사용할 수 있는 자금, 원하는 지역 등을 입력하면 그에 맞는 상권을 분석해서 결과를 예측 or 비슷한 다른 조건들을 추천해주는 시스템 → 고객 맞춤형 보고서 제공
>   
> 필수 기능 : 로그인·회원가입, 게시판글쓰기·글목록
> → 백엔드 작업으로 구현 (**11월 3일까지**) 


- 각자 이름으로 브랜치 만들어서 작업 - main : JOB-A-YO 그대로 이어 작업
